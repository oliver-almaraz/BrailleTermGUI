¡Hola!
este programa es parte de un proyecto sin fines de lucro, además es de código abierto.
Mi finalidad era crear un programa simple que funcionara en casi cualquier plataforma, así que al inicio me
enfoqué en crear un script que se ejecutara desde la terminal sin complicaciones (de ahí el nombre del
programa, 'BrailleTerm'). Sin embargo, no consideré que, al estar escrito en Pyton (3.8), los usuarios de
Windows tendrían que descargar el intérprete.

Estando consciente de que la mayoría de los participantes del curso al que está dedicado este programa
usarían Windows, me tomé el tiempo para hacer esta versión con una interfaz gráfica y lo empaqueté
como un binario  (.exe) para Windows y como un ejecutable para Linux. 

Para ejecutar este programa en Windows, simplemente dale doble click al ejecutable (BrailleTerm*.exe).
En Linux (64bit), si lo mismo no te funciona, deberás darle permisos de ejecución (dándole click derecho /
Propiedades / Permisos / 'Permitir que este archivo se ejcute como programa').
Si quieres ocupar este programa en MacOS o en un sistema diferente, como Linux 32 bit, contáctame.

Una versión para Android y una aplicacción web están actualmente en proceso de desarrollo.

El código fuente de esta versión con interfaz gráfica está disponible en:
https://github.com/oliver-almaraz/BrailleTermGUI
